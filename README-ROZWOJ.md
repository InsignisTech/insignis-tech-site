# Mapa strony INSIGNIS TECH

1. Ekran ładowania (LED/neon, animacje tła)
2. Nawigacja (HOME, O nas, Technologie, Rozwiązania, Firmy, Nasze marki, Kontakt)
3. Hero (nagłówek, claim, hologram, animacje)
4. O nas (misja, opis, statystyki)
5. Technologie (kafle z ikonami, opis technologii)
6. Rozwiązania (kafle branżowe, lista usług)
7. Firmy, które nam zaufały (25 slotów na logo/link)
8. Nasze marki (kafle z opisem i linkiem)
9. Kontakt (dane, formularz z RODO, zero trust)
10. Stopka (linki, social media)

## Pliki do rozbudowy
- index.html — główna struktura i sekcje
- assets/css/styles.css — LED/neon styl, animacje, responsywność
- assets/js/app.js — obsługa menu, formularza, animacji, walidacja RODO
- assets/img/ — logotypy firm, grafiki marek
